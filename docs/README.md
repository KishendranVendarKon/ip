# User Guide

## Features 

### Feature-Adding Tasks to the To-Do-List

1. # todo

   Adds a task with no specific timing

2. # deadline

   Adds a task with a deadline

3. # event
   
   Adds a task that starts at a specified time

### Feature-list

Ability to edit task.

### Feature-Mark as Done

Marks task as done

### Feature-Delete

Deletes task

### Feature-Edit

Edits task
